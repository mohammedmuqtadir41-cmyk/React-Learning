# 🔐 React Login & Signup UI

A modern and responsive **Login & Signup** application built using **React**. This project is part of my React learning journey and is being developed step by step, with new features added as I learn more React concepts.

## 🚀 Features

* 🔑 Login Page
* 📝 Signup Page
* 🎨 Modern and Responsive UI
* ✅ Form Validation
* 📧 Email Validation
* 🔒 Password Validation
* 👁️ Show/Hide Password
* ⚛️ Built with React Functional Components
* 🎯 Controlled Components using `useState`

## 🛠️ Technologies Used

* React
* JavaScript (ES6+)
* JSX
* CSS3
* HTML5

## 💻 Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Navigate to the project folder:

```bash
cd <project-folder>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
